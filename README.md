1. Blog folder - is for uploading a new blog in html format. Once uploaded, update the blogs.json file to pass on the new blog to the webpage.
2. Images folder - contains favicons, og image (for google)
3. nav.css - controls the looks of the website and the dropdown menu.
4. images.json - controls all client images on the website. It links to Wix (as of 7 April 2026).
5. Booking and voucher services are provided by Wix (as of 7 April 2026).
6. Payment handles by Stripe though Wix (as of 7 April 2026).
